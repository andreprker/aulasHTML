Planos de Aula HTML

Guia Interativo de Ensino de HTML

Navegue pelos planos de aula para explorar um currículo prático de desenvolvimento web com HTML. Cada seção oferece objetivos, conteúdo e atividades para uma aula completa.

Plano de Aula 1: Fundamentos do HTML - Estrutura e Conteúdo Básico

Detalhes da Aula

Objetivo: Introduzir a estrutura básica de um documento HTML e os elementos essenciais para criar conteúdo textual.

Duração: 90 minutos

Materiais: Computadores com editor de texto e navegador web.

Conteúdo Programático

  • Introdução ao HTML: O que é e sua função na web.
  • Estrutura Essencial: <!DOCTYPE html>, <html>, <head>, <body>.
  • Meta Tags: <meta charset="UTF-8">, <meta name="viewport" ...>.
  • Título da Página: <title>.
  • Elementos de Texto: <h1>-<h6>, <p>, <br>, <hr>, <strong>, <em>.

Atividade Prática: Minha Primeira Página Pessoal

Os alunos deverão criar um arquivo index.html com os seguintes requisitos:

  • Título da página: "Meu Perfil Online".
  • Um <h1> com o nome completo do aluno.
  • Um <p> com uma breve biografia, utilizando <strong> e <em> para destacar palavras.
  • Um <h2> para "Meus Hobbies".
  • Uma lista de 3 hobbies, cada um em um <p> separado.
  • Um <hr> para separar seções.
  • Um <h3> para "Contato" com um parágrafo contendo um e-mail.

Plano de Aula 2: Links e Imagens - Navegação e Conteúdo Visual

Detalhes da Aula

Objetivo: Capacitar os alunos a criar links para navegação e a inserir imagens em suas páginas web.

Duração: 90 minutos

Materiais: Computadores com editor de texto, navegador web, e algumas imagens de exemplo.

Conteúdo Programático

  • Links (Hyperlinks): Elemento <a>, com atributos href e target="_blank".
  • Imagens: Elemento <img>, com atributos src, alt, width e height.
  • Caminhos Relativos vs. Absolutos.

Atividade Prática: Criando um Álbum de Fotos com Navegação

Criar uma pasta de projeto com index.html, galeria.html, e uma subpasta /imagens.

  • index.html: Deve ter um título, uma introdução e um link para a página galeria.html.
  • galeria.html: Deve exibir três imagens diferentes da pasta /imagens, cada uma com texto alternativo.
  • Navegação: A página da galeria deve ter um link para retornar à página inicial.

Plano de Aula 3: Listas e Tabelas - Organizando Informações

Detalhes da Aula

Objetivo: Ensinar a estruturar informações de forma organizada usando listas e tabelas HTML.

Duração: 90 minutos

Materiais: Computadores com editor de texto e navegador web.

Conteúdo Programático

  • Listas não ordenadas: <ul> e <li>.
  • Listas ordenadas: <ol> e <li>.
  • Listas aninhadas.
  • Tabelas: <table>, <thead>, <tbody>, <tr>, <th>, <td>.
  • Mesclagem de Células: colspan e rowspan.

Atividade Prática: Currículo Simplificado e Horário de Aulas

Criar um arquivo curriculo.html.

  • Seção de Habilidades: Usar listas aninhadas (<ul> e <ol>) para listar habilidades técnicas e comportamentais.
  • Seção de Horário: Usar uma <table> para criar um horário de aulas semanal.

Plano de Aula 4: Formulários - Coletando Dados do Usuário

Detalhes da Aula

Objetivo: Ensinar a criar formulários HTML para coletar diferentes tipos de dados dos usuários.

Duração: 90 minutos

Materiais: Computadores com editor de texto e navegador web.

Conteúdo Programático

  • Elemento <form> com atributos action e method.
  • Inputs variados: text, password, email, number, date, checkbox, radio, submit, reset.
  • Outros elementos: <label>, <textarea>, <select>, <option>, <button>.
  • Atributos importantes: name, id, placeholder, value, required.

Atividade Prática: Formulário de Contato ou Cadastro

Criar um arquivo contato.html com um formulário que colete nome, e-mail, mensagem, assunto (radio), interesses (checkbox) e um país (select).

Garantir que todos os campos tenham um <label> associado para acessibilidade.

Plano de Aula 5: Elementos Semânticos e Multimídia

Detalhes da Aula

Objetivo: Ensinar a usar elementos HTML semânticos para uma estrutura mais significativa e a incorporar áudio e vídeo.

Duração: 90 minutos

Materiais: Computadores, navegador web, e arquivos de áudio/vídeo pequenos.

Conteúdo Programático

  • HTML Semântico: <header>, <nav>, <main>, <article>, <section>, <aside>, <footer>.
  • Multimídia: Elementos <audio> e <video> com o atributo controls.
  • Incorporação externa: Uso de <iframe> para vídeos do YouTube.

Atividade Prática: Criando um "Blog Post" Semântico com Mídia

Criar um arquivo blog_post.html utilizando toda a estrutura semântica (header, nav, main, article, footer).

Dentro do <article>, incluir texto, uma imagem com <figure> e <figcaption>, e um vídeo incorporado do YouTube.

Plano de Aula 6: Boas Práticas, Validação e Meta Tags

Detalhes da Aula

Objetivo: Reforçar boas práticas, ensinar a validar HTML e explorar meta tags para otimização.

Duração: 90 minutos

Materiais: Computadores com acesso à internet para o W3C HTML Validator.

Conteúdo Programático

  • Boas Práticas: Indentação, comentários, nomenclatura.
  • Validação HTML: O porquê e como usar o W3C HTML Validator.
  • Meta Tags para SEO: description.
  • Meta Tags para Redes Sociais: Open Graph (og:title, og:description, etc.).
  • Favicon: <link rel="icon" ...>.

Atividade Prática: Otimizando e Validando uma Página Existente

Pegar uma página criada anteriormente e aplicar as otimizações:

  • Revisar e indentar o código.
  • Adicionar comentários explicativos.
  • Validar o código no W3C Validator e corrigir todos os erros.
  • Adicionar uma meta tag de descrição e uma meta tag Open Graph para o título.
