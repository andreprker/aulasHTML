Estrutura Essencial do HTML

Descubra os elementos fundamentais que compõem cada documento HTML e a sua função na web.

<!DOCTYPE html>

Esta é a primeira linha de qualquer documento HTML moderno.

  • O que é: Não é uma tag HTML, mas uma "declaração de tipo de documento".
  • Função: Informa ao navegador qual versão do HTML está a ser utilizada. <!DOCTYPE html> especifica que o documento é um HTML5, a versão mais recente e padrão. Isso garante que o navegador renderize a página no "modo padrão", evitando comportamentos inesperados do "modo quirks" (modo de compatibilidade para navegadores mais antigos). 
<!DOCTYPE html>
<html>
    <!-- Restante do código HTML -->
</html>

<html>

O elemento raiz que encapsula todo o conteúdo HTML de uma página.

  • O que é: A tag raiz de todo documento HTML.
  • Função: É o elemento que engloba todo o conteúdo da página web, exceto a declaração <!DOCTYPE html>. Tudo o que faz parte da página HTML deve estar dentro desta tag. O atributo lang="pt-BR" (ou outro código de idioma) dentro da tag <html> é importante para acessibilidade e SEO, indicando o idioma principal do conteúdo. 
<!DOCTYPE html>
<html lang="pt-BR">
    <head>
        <!-- Metadados -->
    </head>
    <body>
        <!-- Conteúdo visível -->
    </body>
</html>

<head>

A secção de cabeçalho, que contém metadados sobre a página.

  • O que é: A secção de cabeçalho do documento HTML.
  • Função: Contém metadados sobre a página web que não são visíveis diretamente no corpo da página. Isso inclui:
    • <meta charset="UTF-8">: Define a codificação de caracteres, garantindo que os textos sejam exibidos corretamente (incluindo acentos e caracteres especiais).
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Essencial para a responsividade, ajustando a visualização da página em diferentes dispositivos.
    • <title>: Define o título da página que aparece na aba do navegador.
    • <link>: Para ligar a folhas de estilo CSS externas.
    • <script>: Para incluir ficheiros JavaScript ou código JavaScript interno.
    • Outras tags <meta> para descrição, palavras-chave, etc.
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Minha Página</title>
    <link rel="stylesheet" href="estilos.css">
    <script src="script.js" defer></script>
</head>

<body>

A secção que contém todo o conteúdo visível da página web.

  • O que é: A secção do corpo do documento HTML.
  • Função: Contém todo o conteúdo visível da página web. Tudo o que o utilizador vê no navegador (textos, imagens, links, vídeos, tabelas, formulários, etc.) é colocado dentro desta tag. 
<body>
    <h1>Bem-vindo à Minha Página</h1>
    <p>Este é um parágrafo de texto.</p>
    <img src="imagem.jpg" alt="Descrição da imagem">
    <a href="outra-pagina.html">Ir para outra página</a>
</body>

Esses quatro elementos formam a estrutura básica e essencial que todo documento HTML deve ter para ser válido e bem interpretado pelos navegadores, sendo a base para a construção de qualquer página web.
