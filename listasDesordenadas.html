Listas Não Ordenadas: <ul> e <li> em HTML

As listas são elementos HTML essenciais para organizar o conteúdo de forma clara e legível. Existem dois tipos principais de listas: ordenadas (<ol>) e não ordenadas (<ul>). Esta página focará nas listas não ordenadas, que são usadas para agrupar itens relacionados onde a ordem não é semanticamente importante.

---

O Elemento <ul> (Unordered List)

O elemento <ul> representa uma lista de itens em que a ordem dos itens não tem significado. Por padrão, os navegadores exibirão os itens de uma lista não ordenada com marcadores (geralmente círculos sólidos ou discos).

A tag de abertura <ul> e a tag de fechamento </ul> delimitam toda a lista.

<ul>
    <li>Item da lista 1</li>
    <li>Item da lista 2</li>
</ul>
---

O Elemento <li> (List Item)

O elemento <li> representa um item individual dentro de uma lista (ordenada <ol> ou não ordenada <ul>). Cada item que você deseja incluir na sua lista deve ser envolvido por uma tag <li> de abertura e uma tag </li> de fechamento.

É importante notar que o elemento <li> deve ser um filho direto de um <ul> ou <ol>.

---

Exemplo Prático de Lista Não Ordenada

Vamos criar uma lista simples de compras para ilustrar o uso de <ul> e <li>:

Código HTML:

<h2>Minha Lista de Compras</h2>
<ul>
    <li>Pão</li>
    <li>Leite</li>
    <li>Ovos</li>
    <li>Frutas</li>
</ul>

Visualização:

Minha Lista de Compras

  • Pão
  • Leite
  • Ovos
  • Frutas
---

Estilizando Listas com CSS

Embora as listas não ordenadas venham com um estilo padrão (marcadores), você pode alterar a aparência desses marcadores usando CSS. A propriedade list-style-type é comumente usada para isso.

Exemplo de Estilização CSS:

<h2>Tarefas do Dia (Estilo Personalizado)</h2>
<ul class="custom-style">
    <li>Estudar HTML</li>
    <li>Fazer exercícios</li>
    <li>Caminhar com o cachorro</li>
</ul>

<!-- No CSS (dentro da tag <style> ou em um arquivo .css): -->
<style>
    .custom-style {
        list-style-type: square; /* Marcadores quadrados */
        color: #663399; /* Cor do texto e dos marcadores */
    }
</style>

Visualização (com o CSS acima aplicado):

Tarefas do Dia (Estilo Personalizado)

  • Estudar HTML
  • Fazer exercícios
  • Caminhar com o cachorro

Listas Aninhadas

Você pode aninhar listas, o que significa colocar uma lista dentro de outra. Isso é útil para criar estruturas hierárquicas de informações.

Exemplo de Lista Aninhada:

<h2>Menu de Comida</h2>
<ul>
    <li>Entradas
        <ul>
            <li>Pão de alho</li>
            <li>Salada Ceasar</li>
        </ul>
    </li>
    <li>Pratos Principais
        <ul>
            <li>Macarrão com frutos do mar</li>
            <li>Bife à parmegiana</li>
        </ul>
    </li>
    <li>Sobremesas
        <ul>
            <li>Pudim</li>
            <li>Bolo de chocolate</li>
        </ul>
    </li>
</ul>

Visualização:

Menu de Comida

  • Entradas
    • Pão de alho
    • Salada Ceasar
  • Pratos Principais
    • Macarrão com frutos do mar
    • Bife à parmegiana
  • Sobremesas
    • Pudim
    • Bolo de chocolate
---

Conclusão

O uso dos elementos <ul> e <li> é fundamental para estruturar conteúdo não sequencial em suas páginas web. Eles não apenas melhoram a legibilidade para os usuários, mas também fornecem significado semântico para motores de busca e tecnologias assistivas. Com um pouco de CSS, você pode personalizá-los para se adequarem perfeitamente ao design do seu site.
