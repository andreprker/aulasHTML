Imagens na Web: O Elemento `<img>`

As imagens são um componente essencial de praticamente qualquer página web, tornando o conteúdo mais visualmente atraente e informativo. Em HTML, o elemento usado para incorporar imagens é o <img>.

O elemento <img> é um elemento "vazio" (ou seja, não possui uma tag de fechamento como </img>) e seus detalhes são definidos através de atributos.

---

Atributos Essenciais do `<img>`

Para que uma imagem seja exibida corretamente e de forma acessível, é fundamental entender e utilizar seus atributos principais:

  • src (Source - Fonte): Este é o atributo mais crucial. Ele especifica o caminho (URL) para o arquivo da imagem. Sem o atributo src, o navegador não saberá qual imagem carregar.
    <img src="caminho/para/minha-imagem.jpg">
  • alt (Alternative Text - Texto Alternativo): Este atributo fornece uma descrição textual da imagem. É extremamente importante por várias razões:
    • Acessibilidade: Leitores de tela para usuários com deficiência visual leem o texto alt, permitindo que eles compreendam o conteúdo da imagem.
    • SEO (Otimização para Motores de Busca): Motores de busca usam o texto alt para entender o conteúdo da imagem, o que pode ajudar no ranqueamento.
    • Fallback: Se a imagem não puder ser carregada (por um link quebrado, problema de conexão, etc.), o texto alt será exibido em seu lugar.
    <img src="por-do-sol.jpg" alt="Um pôr do sol vibrante sobre o oceano com cores quentes no céu">
  • width (Largura): Este atributo especifica a largura da imagem em pixels. Definir a largura e a altura pode ajudar o navegador a reservar o espaço necessário para a imagem antes mesmo dela carregar, evitando "saltos" no layout.
    <img src="logo.png" alt="Logotipo da empresa" width="150">
  • height (Altura): Semelhante ao width, este atributo especifica a altura da imagem em pixels. Se você definir apenas um dos atributos (largura ou altura), o navegador ajustará o outro proporcionalmente para manter a proporção original da imagem, a menos que você sobrescreva esse comportamento com CSS.
    <img src="banner.jpg" alt="Banner promocional" width="800" height="200">

Boas Práticas e Considerações

  • Sempre inclua o atributo alt para garantir acessibilidade e SEO.
  • Para controle de layout responsivo, é geralmente melhor usar CSS para definir width e height (por exemplo, img { max-width: 100%; height: auto; }) em vez dos atributos HTML, especialmente se você precisar que suas imagens se ajustem a diferentes tamanhos de tela.
  • Use imagens otimizadas para a web (com o tamanho de arquivo menor possível sem perder qualidade) para melhorar o tempo de carregamento da página.
  • Considere usar diferentes formatos de imagem (JPEG para fotos, PNG para gráficos com transparência, SVG para gráficos vetoriais escaláveis).

Dominar o uso do elemento <img> e seus atributos é fundamental para criar páginas web ricas em conteúdo visual e acessíveis.
