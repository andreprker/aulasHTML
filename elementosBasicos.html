Título da Página e Elementos Essenciais de Texto HTML

Aprenda a usar as tags HTML mais comuns para estruturar e formatar o conteúdo textual da sua página web.

Título da Página: <title>

  • Função: Define o título da página web. Este título não é visível no corpo da página, mas aparece na aba do navegador, na barra de título da janela ou como o nome predefinido quando a página é adicionada aos favoritos. É crucial para a usabilidade e para o SEO (Search Engine Optimization). 
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>O Meu Incrível Site</title> <!-- Este é o título -->
</head>

Elementos de Texto

Estes elementos são a base para organizar e estilizar o texto dentro do corpo da sua página web.

<h1> a <h6>

São tags de cabeçalho (ou títulos de secção). <h1> representa o título mais importante da página (geralmente há apenas um por página), enquanto <h6> representa o menos importante. Eles são usados para estruturar o conteúdo e indicar hierarquia, o que é importante para a legibilidade e para os motores de busca. 

<body>
    <h1>Título Principal da Página</h1>
    <h2>Secção 1</h2>
    <h3>Sub-secção 1.1</h3>
</body>

<p>

Define um parágrafo de texto. É a tag mais comum para blocos de texto simples numa página web. O navegador adiciona automaticamente algum espaçamento antes e depois de cada parágrafo. 

<p>Este é um exemplo de um parágrafo de texto.</p>
<p>Aqui está outro parágrafo com mais informações.</p>

<br>

Cria uma quebra de linha simples. Ao contrário do <p>, que cria um novo parágrafo com espaçamento, o <br> apenas avança para a próxima linha sem adicionar espaçamento extra. É uma tag vazia (não tem tag de fecho). 

Linha de texto um.<br>
Linha de texto dois.

<hr>

Representa uma "linha horizontal" ou uma "mudança temática" no conteúdo. É frequentemente renderizado como uma linha divisória visual, mas o seu propósito semântico é indicar uma separação de conteúdo. É uma tag vazia. 

<p>Primeiro bloco de conteúdo.</p>
<hr> <!-- Insere uma linha divisória -->
<p>Segundo bloco de conteúdo.</p>

<strong>

É uma tag de formatação de texto que indica que o conteúdo dentro dela tem uma forte importância ou relevância. Os navegadores geralmente renderizam o texto em **negrito**, mas o seu significado semântico é mais do que apenas visual. 

<p>Este é um texto <strong>muito importante</strong> para ler.</p>

<em>

É uma tag de formatação de texto que indica que o conteúdo dentro dela deve ser enfatizado. Os navegadores geralmente renderizam o texto em *itálico*, mas, tal como <strong>, o seu significado é semântico, indicando ênfase no discurso. 

<p>Eu <em>realmente</em> adoro programar em HTML.</p>

Dominar estas tags é o primeiro passo para criar conteúdo web bem estruturado e significativo.
