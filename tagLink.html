Links (Hyperlinks): O Coração da Navegação na Web

Explore o elemento essencial para conectar páginas na internet: o <a>.

O Elemento <a>: Âncora

O elemento HTML <a> (de "âncora") é usado para criar hyperlinks para outras páginas web, arquivos, locais dentro da mesma página, endereços de e-mail ou qualquer outra URL. É o que torna a web interconectada!

A estrutura básica de um link é: 

<a href="url_do_destino">Texto do Link</a>

Atributo href: O Destino do Link

O atributo href (de "hypertext reference") é o mais importante para o elemento <a>. Ele especifica a URL para a qual o link aponta.

Você pode usar URLs absolutas (para sites externos) ou URLs relativas (para páginas dentro do seu próprio site).

Exemplos de href:

  • Link para um site externo: Visite o Google 
    <a href="https://www.google.com">Visite o Google</a>
  • Link para uma página interna (exemplo, se houvesse uma página 'sobre.html'): Sobre Nós (link de exemplo) 
    <a href="sobre.html">Sobre Nós</a>
  • Link para um endereço de e-mail: Envie um E-mail 
    <a href="mailto:exemplo@dominio.com">Envie um E-mail</a>

Atributo target="_blank": Abrindo em Nova Aba

O atributo target especifica onde o documento vinculado será aberto. O valor _blank é amplamente utilizado para abrir o link em uma nova aba ou janela do navegador. Isso é útil para que o usuário não saia do seu site atual.

É uma boa prática usá-lo para links externos, mantendo o usuário na sua página.

Exemplos de target="_blank":

  • Link para o site da Wikipedia (abrirá em nova aba): Visitar Wikipedia (Nova Aba) 
    <a href="https://pt.wikipedia.org" target="_blank">Visitar Wikipedia (Nova Aba)</a>
  • Link para a documentação MDN (abrirá em nova aba): MDN Docs sobre <a> (Nova Aba) 
    <a href="https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/a" target="_blank">MDN Docs sobre <a> (Nova Aba)</a>

Dica de Segurança:

Ao usar target="_blank", é recomendado adicionar rel="noopener noreferrer" ao seu link para prevenir vulnerabilidades de segurança e melhorar o desempenho. Exemplo: <a href="..." target="_blank" rel="noopener noreferrer">.
